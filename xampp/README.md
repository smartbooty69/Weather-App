# Data Structures Programming Lab

## Master of Computer Applications (MCA)

---

### Getting Started

Welcome to the Data Structures Programming Lab for the Master of Computer Applications program at St. Joseph’s University. This repository contains a collection of programming assignments and projects focused on various data structures using the C programming language. Follow the steps below to set up your environment and start working on the lab assignments.

#### Prerequisites

Before you start, ensure you have the following installed on your system:

- **C Compiler:** You can use GCC (GNU Compiler Collection) or any other C compiler suitable for your operating system. For Windows, you may consider installing MinGW or using an IDE like Code::Blocks, Dev-C++, or Visual Studio.

- **Text Editor/IDE:** Use a text editor or Integrated Development Environment (IDE) of your choice for writing and editing your C code. Some popular options include:
  - **Visual Studio Code**
  - **Code::Blocks**
  - **Dev-C++**
  - **Eclipse CDT**

- **Basic Knowledge of C Programming:** Familiarity with C syntax, functions, and control structures will be beneficial for completing the assignments.

#### Cloning the Repository

To get started with the Data Structures lab projects, clone the repository to your local machine. Open your terminal or command prompt and run:

```bash
git clone https://github.com/smartbooty69/MCA-2024-2026.git
```

#### Project Structure

The projects are organized into directories based on their topics. Navigate to the **Data Structures Programming Lab** folder to access the individual programming assignments. Each project is named sequentially (e.g., `Prog01.c`, `Prog02.c`, etc.) and contains the necessary C code for implementing various data structures and algorithms.

#### Running the Projects

1. **Open Your IDE or Text Editor:**
   - Launch your chosen IDE or text editor.

2. **Navigate to the Project Directory:**
   - Open the folder containing the Data Structures lab files.

3. **Compile the C Program:**
   - Open the desired C program file (e.g., `Prog01.c`).
   - Compile the code using the terminal or IDE options. If using GCC, you can compile the program using the following command:
     ```bash
     gcc Prog01.c -o Prog01
     ```

4. **Run the Compiled Program:**
   - After successful compilation, run the program using the command:
     ```bash
     ./Prog01
     ```
   - (Use `Prog01.exe` for Windows)

5. **Verify Output:**
   - Check the console for output and any other results generated by your program. Modify the input or logic as necessary based on your assignment requirements.

#### Contributing

Contributions to this repository are welcome. If you have enhancements or additional assignments, feel free to fork the repository and submit pull requests. Ensure to follow best coding practices and comment your code for clarity.
